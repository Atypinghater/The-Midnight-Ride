Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Gameplay into the pop-up and select OK

Equilibrium

Install normally

- Mass weapon balance project for both the base game and the DLCs

Better Explosives

Install normally

- Rebalances the damage and range of explosives, as well as other other balance tweaks

Unofficial Fallout 4 Patch - Better Explosives Patch

Install normally

- Forwards fixes from UF4P that were overwritten by Better Explosives

Encounter Zone Recalculation

Installation instructions:

  • Download the main file manually
  • Extract the F4SE folder from the archive to the game's Data folder

- Allows previously-visited encounter zones to update the levels of the enemies in them as the player levels up

Decreased Automatron Encounters

Installation instructions:

  • Download the main file normally through MO2
  • Once the mod is installed, run FO4Edit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to RustDevilRedux.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Expand the RustDevilRedux.esp header in the left pane
  • Right-click on the Non-Player Character (Actor) header and select Remove
    • ×
    • Removes unnecessary changes to NPCs that cause conflicts with Unofficial Patch and Unbogus Health Ranges
  • Press control+s to save the changes, and then exit the program

- Lowers the rate of Automatron's random encounters to be that of vanilla random encounters

Loot Logic and Reduction

Files to install normally:

  • Main file - Loot Logic and Reduction 1.5.3.1
  • Optional file - Loot Logic Scrounger and Fortune Finder v2

- Reduces the amount of valuable loot and diversifies the type of loot in generic containers

Complex Vendors

Install normally

- Makes most vendors only buy items related to what they sell (Doctors only buy medical supplies, farmers only buy food, etc.)

Lore-Based Power Armor Changes

Install normally

- Balances Power Armor by making it more difficult to acquire and maintain

RobCo Recon Scope Replacer

Install normally

- Gives Recon Scopes a much sleeker and less obnoxious look

Salvage Beacons

Install normally

- Adds a lore friendly and balanced way to send loot back to settlements while exploring
NOTE: Make sure you read the description to know how to craft and use the beacons
