Gameplay

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Gameplay.
It is especially important to read the descriptions of these mods.

True Invisibility Redux

Installation instructions:

  • Main Files - True Invisibility 3.2 FOMOD
  • You will be prompted with a FOMOD installer, click Install
Completely overhauls the invisibility system, all fully configurable via an MCM.

Better Power Armor - Redux

Installation instructions:

  • Main Files - Better Power Armor - Redux
  • Better Power Armor - Redux - FATE Fixes Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Does multiple things:
  • Your Power Armor will now fully absorb the damage of ballistic and laser weapon types.
  • Each armor part will work individually and absorb the damage of the body part it covers.
  • When one power armor part breaks, only this body part that was covered will take damage.

Thrive Redone

Installation instructions:

  • Main Files - Thrive Redone
Fixes balance issues with the original game and makes the survival aspect of it more challenging, removing shortcuts and silly mechanics.

Power Armor Repair Kits

Installation instructions:

  • Main Files - Kat's Power Armor Repair Kits
  • You will be prompted with a FOMOD installer, click next then Install
Changes the system for repairing power armor to use repair kits craftable at the chemistry station.

Encounter Zone Recalculation

Installation instructions:

  • Main Files - Encounter Zone Recalculation
Allows previously-visited encounter zones to update the levels of the enemies in them as the player levels up.

Complex Vendors

Installation instructions:

  • Main Files - Complex Vendors
  • Complex Vendors - UFO4P Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  • Complex Vendors - FATE Fixes Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Makes most vendors only buy items related to what they sell (for example doctors will only buy medical supplies).
