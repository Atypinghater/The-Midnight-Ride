Gameplay

Creating a Separator in MO2
  1. Right-click the empty space in the left pane of MO2 and select Create Separator
  2. Name the separator Gameplay

It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Thrive Redone

Installation instructions:

  • Main File - Thrive Redone
  • You will be prompted with a FOMOD installer, click Install

- Fixes balance issues with the original game and makes the survival aspect of it more challenging, removing shortcuts and silly mechanics

Some Assembly Required

Installation instructions:

  1. Main File - Some Assembly Required

- Manually removes nearly every Power Armor spawn in the game and makes Power Armor exceedingly rare and difficult to acquire. Just as it should be

Power Armor Repair Kits

Installation instructions:

  1. Main File - Kat's Power Armor Repair Kits
  2. You will be prompted with a FOMOD installer, click Next and then Install

- Changes the system for repairing power armor to use repair kits craftable at the chemistry station

Encounter Zone Recalculation

Installation instructions:

  • Main File - Encounter Zone Recalculation

- Allows previously-visited encounter zones to update the levels of the enemies in them as the player levels up

Decreased Automatron Encounters

Installation instructions:

  • Main File - Rust Devil Redux

- Lowers the rate of Automatron's random encounters to be that of vanilla random encounters

Complex Vendors

Installation instructions:

  • Main File - Complex Vendors
  • Also get the UFO4P Patch here

- Makes most vendors only buy items related to what they sell (doctors only buy medical supplies, farmers only buy food, etc.)

Salvage Beacons

Installation instructions:

  • Main File - Salvage Beacons

- Adds a lore friendly and balanced way to send loot back to settlements while exploring

Make sure you read the description to know how to craft and use the beacons

True Invisibility

Installation instructions:

  1. Main File - True Invisibility 3.2 FOMOD
  2. You will be prompted with a FOMOD installer, click Install

- Completely overhauls the invisibility system, all fully configurable via an MCM
