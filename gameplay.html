Gameplay

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Gameplay

It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

Lunar Fallout Overhaul

Installation instructions:

  • Main File - LunarFalloutOverhaul ver100

- Overhauls the entire game from the ground up to create a more difficult and rewarding survival experience

This mod will make the game much harder. You may want to skip this mod if you are a beginner.

Power Armor Repair Kits

Installation instructions:

  1. Download the Main File - Kat's Power Armor Repair Kits 1.3.1 and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should just select Next then Install

- Changes the system for repairing power armor to use repair kits craftable at the chemistry station

Encounter Zone Recalculation

Installation instructions:

  • Main File - Encounter Zone Recalculation 1.0

- Allows previously-visited encounter zones to update the levels of the enemies in them as the player levels up

Decreased Automatron Encounters

Installation instructions:

  • Main File - Rust Devil Redux .1

- Lowers the rate of Automatron's random encounters to be that of vanilla random encounters

Complex Vendors

Installation instructions:

  • Main File - Complex Vendors 0.9.1

- Makes most vendors only buy items related to what they sell (doctors only buy medical supplies, farmers only buy food, etc.)

Salvage Beacons

Installation instructions:

  • Main File - Salvage Beacons 1.0.4

- Adds a lore friendly and balanced way to send loot back to settlements while exploring

Make sure you read the description to know how to craft and use the beacons

True Invisibility

Installation instructions:

  1. Download the Main File - True Invisibility 3.2 FOMOD and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should just select Install

- Completely overhauls the invisibility system, all fully configurable via an MCM

Survival Saving Holotapes

Installation instructions:

  1. Download the Main File - Survival Saving - Holotapes 1.6 and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should just select Install

- Adds an immersive and balanced way of saving in survival mode, through special holotapes found in loot
