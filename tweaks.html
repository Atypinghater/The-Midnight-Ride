Tweaks

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Tweaks.

Power Armor Ownership Fix

Installation instructions:

  • Optional File - PA Ownership Fix (with No PA Battery Pathing)
When you take over a Power Armor that belonged to someone else and insert a fusion core, it will no longer be considered stolen. Also stops you from having to path all the way around power armor when putting in a battery, as this can often result in getting stuck in terrain.

Fortune Finder 4 Fix

Installation instructions:

  • Main File - Fortune Finder 4 Fix - Basic Bottlecap Mine
Changes the fourth rank of Fortune Finder to put the generated caps directly into your inventory rather than scattered around the game world, as all the loose objects can be hard to collect and can accelerate save-bloating.

Raw Input

Installation instructions:

  1. Main File - RAW INPUT - NMM Installer
  2. You will be prompted with a FOMOD installer, where you should select the ADS sensitivity percent you prefer
    • This is completely up to personal preference, but I use the 100% option
Disables mouse acceleration, normalizes sensitivity, and more input-related fixes.

FOV Slider and Player Height

Installation instructions:

  • Main File - FOV Slider and Player Height
Adds a MCM that allows you to configure all types of FOV and player height options.

Unlimited Survival Mode

Installation instructions:

  • Main File - Unlimited Survival Mode - F4SE
Restores the game features disabled in Survival mode, such as fast travel and saving. Fully configurable.

LOST Audio Tweaks

Installation instructions:

  • Main File - LOST Audio Tweaks
  • Main File - LOST Audio Tweaks - No RAO Replacer (select the name that matches the file on Nexus when the installation prompt comes up)
Make sure the No RAO Replacer mod is loaded below the main one.
Fixes many audio issues and annoyances.

Keep Radiants in the Commonwealth UFO4P Version

Installation instructions:

  • Main File - Keep Radiants In The Commonwealth UFO4P Version
Prevents radiant quests from being generated in DLC areas.

Who's The General

Installation instructions:

  • Main File - Who's The General - Minutemen Quest Cleanup
  • Also get the UFO4P patch here (select the name that matches the file on Nexus when the installation prompt comes up)
  • Optional File - 1 - Keep Radiants in the Commonwealth Patch (select the name that matches the file on Nexus when the installation prompt comes up)
A mod for players who dislike being asked to report back to Preston Garvey.

Bullet Counted Reload System

Installation instructions:

  • Main File - Bullet Counted Reload - v2.00
  • Optional File - BCR- Lever Action Rifle v1.02 (select the name that matches the file on Nexus when the installation prompt comes up)
Allows reloading the lever-action rifle based on the amount of shots fired.

Whose Quest Is It Anyway

Installation instructions:

  • Main File - Whose Quest Is It Anyway
Swap the "quest items can't be removed" message with something a little more informative.
