Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Finish Line into the pop-up and select OK

Load Order.txt

Installation instructions:

  • Extract loadorder.txt from the download archive to Mod Organizer 2/profiles/The Midnight Ride and overwrite when prompted
  • Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

- Automatically ensures your load order is correct for the guide. This will still work if you didn't install certain mods, but if you added your own mods they will be sorted to the bottom of the load order and should be manually moved back to their proper position

Ruddy88's Simple Sorter

Installation instructions:

  • Download the R88SimpleSorter_XML main file normally through MO2
  • Download the R88SimpleSorter_SCRIPT main file manually
  • From the R88SimpleSorter_SCRIPT archive, extract the Edit Scripts folder to where you have FO4Edit installed
    • The Edit Scripts from the archive should align with the Edit Scripts folder already in the FO4Edit folder
  • Once the folder is extracted, run FO4Edit through MO2
  • At the Module Selection pop-up, just hold down left shift and click OK
  • At the ModGroups pop-up, just select OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Right-click anywhere in the right pane and select Apply Script...
  • In the pop-up, select R88_SimpleSorter from the Script drop-down then select OKs
  • In the Options pop-up, select the following options then select New Patch:
    • ×
  • In the next pop-up, make sure every plugin is checked then select OK
  • In the next pop-up, name the file Ruddy's Simple Sorter then select OK
  • Click OK again to let the script run (may take a few minutes)
  • Wait for the message Done: Applying script "R88_SimpleSorter" to appear in the right pane
  • Press control+s to save the changes, and then exit the program
  • Once you have exited the program, double-click on the Overwrite mod at the bottom of the left pane of MO2
  • Drag Ruddy's Simple Sorter.esp from the Overwrite to the Ruddy88's Simple Sorter mod in the left pane of MO2
  • Check Ruddy's Simple Sorter.esp in the right pane of MO2

- Adds an automatically-compatible item sorting method

Configuring The Mod Configuration Menu

The MCM can be accessed through the pause menu in-game. Any settings you change will be consistent across all saves

  • Crafting Highlight Fix: Leave default
  • Faster Workshop: Adjust to personal preference (I personally use the B key)
  • Better Explosives Redux: Adjust to personal preference (I personally keep all the settings default)
  • XDI: Adjust to personal preference (I personally disable both Dialogue Cues options

Congratulations!

Congratulations, you are now done with your modding journey! Remember to stop by the Discord if you have any questions/comments/concerns. One thing to note, it is only recommended to add your own mods if you are an experienced modder and can figure out what is/isn't compatible and what is safe or not. Fallout 4 can be very fickle, especially with the pre-combined meshes and the save-baked papyrus scripts.
