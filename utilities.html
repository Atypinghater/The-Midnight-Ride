A Note About Testing the Game

After each section in every module, you should launch the game and load a save/start a new game just to make sure the game loads properly.

Fallout 4 Script Extender

Installation instructions:

  • Click the topmost download link next to Fallout 4 runtime 1.10.163 - build: 0.6.21
  • Extract the 2 files from the archive ending with .dll and the file ending with .exe to the game's Root folder
    • If you do not know what the Root folder is, read the Common Terminology section from the Installation page

- Extends the scripting capabilities Fallout 4
NOTE: In order for the script extender to work, you need to launch the game using the f4se_loader.exe file. It will automatically be added to Mod Organizer 2 upon restarting it

F4SE Plugin Preloader

Installation instructions:

  • Download the main file manually
  • Extract the two files from the downloaded archive to the game's Root folder
  • Once the files have been extracted, open xSE PluginPreloader.xml
  • Change <LoadMethod Name="OnProcessAttach"> (line 26) to <LoadMethod Name="OnThreadAttach">

- Allows F4SE plugins to be loaded before the game initializes

Buffout 4

Installation instructions:

  • Download both main files manually
  • From the Buffout 4 archive, extract the F4SE folder to the game's Data folder
  • From the TBB Redistributables archive, extract the tbbmalloc.dll file to the Root folder

- Fixes many bugs with the game's engine
NOTE: The reason that these F4SE plugins are installed directly into the data folder and not through Mod Organizer 2 is that MO2's virtual file system can rarely fail to hook the .DLLs in time when launching the game, and therefore they will not work in-game (and this problem can be magnified by F4SE Plugin Preloader)

High FPS Physics Fix

Installation instructions:

  • Download the High FPS Physics Fix main file manually
  • Extract the F4SE folder from the archive to the game's Data folder
  • Once the folder has been extracted, navgiate to Data/F4SE/Plugins and open HighFPSPhysicsFix.ini
  • In the .INI, change the following:
    • Set DisableBlackLoadingScreens to true (line 13)
    • Set DisableAnimationOnLoadingScreens to true (line 17)

- Allows for playing at framerates over 60 and reduces load times

Sprint Stuttering Fix

Installation instructions:

  • Download the main file manually
  • Extract the F4SE folder from the archive to the game's Data folder

- Fixes camera stuttering while sprinting on uneven surfaces

Faster Workshop

Installation instructions:

  • Download the main file manually
  • Extract the Data folder from the archive to the game's Root folder

- Speeds up the time it takes to open the workshop menu, especially if you have mods that add settlement items
