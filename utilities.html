Utilities

Fallout 4 Script Extender

Installation instructions:

  1. Click the topmost download link next to Fallout 4 runtime 1.10.163 - build: 0.6.21
  2. From the downloaded archive, extract all the files to the game's Root folder
If you do not know what the Root folder is, read the Key Terminology section from the Initial Setup page.

- Extends the scripting capabilities of the game

In order for the script extender to work, you need to launch the game using the f4se_loader.exe file. It will automatically be added to Mod Organizer 2 upon restarting it.

PluginPreloader

Installation instructions:

  1. Download the Main File - xSE PluginPreloader F4 0.2.4 using the Manual Download button
  2. From the download archive, extract the two files to the game's Root folder
  3. Once the files have been extracted, open xSE PluginPreloader.xml in a text editor
  4. Change <LoadMethod Name="OnProcessAttach"> (line 14) to <LoadMethod Name="OnThreadAttach">

- Allows F4SE plugins to be loaded before the game initializes

Address Library for F4SE Plugins

Installation instructions:

  • Main File - Address Library 1.10.163.0
If you do not know what these instructions mean, or how to install mods with MO2, read the Mod Installation Advice section from the MO2 page

- Resource required for other F4SE plugins

PrivateProfileRedirector

Installation instructions:

  • Main File - PrivateProfileRedirector F4 0.5

- Speeds up the launch process by storing INI files in memory

Buffout 4

Installation instructions:

  1. Download and install the Main File - Buffout 4 1.20.3 normally through MO2
  2. Download the Main File - TBB Redistributables 2020.3 file using the Manual Download button
  3. From the downloaded archive, extract tbbmalloc.dll to the game's Root folder

- Fixes numerous bugs within the game's engine

High FPS Physics Fix

Installation instructions:

  1. Download and install the Main File - High FPS Physics Fix 0.7.7 normally through MO2
  2. Once the mod has been installed, double-click on it in the left pane of MO2
  3. In the INI-Files tab, select F4SE/Plugins/HighFPSPhysicsFix.ini
  4. Change the following options:
    • Set DisableBlackLoadingScreens= to true (line 36)
    • Set DisableAnimationOnLoadingScreens= to true (line 41)

- Allows for playing at framerates over 60, mitigates stutter, and reduces load times

Sprint Stutter Fix

Installation instructions:

  • Main File - Sprint Stuttering Fix 1.2 (the top one)

- Stops the camera from stuttering while sprinting on uneven surfaces

Crafting Highlight Fix

Installation instructions:

  • Main File - Crafting Highlight Fix 1.8.8
