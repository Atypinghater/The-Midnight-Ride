Utilities

Make sure the The Midnight Ride profile is active before proceeding.

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Utilities.

F4SE

Installation instructions:

  1. Main Files - Fallout 4 Script Extender (F4SE) (Manual Download)
  2. From the downloaded archive, extract the contents of the archive to the game's Root folder
If you do not know what the Root folder is, read the Key Terminology section from the Initial Setup page.
You need to extract all of the files and folders contained in the archive.

PluginPreloader

Installation instructions:

  1. Download the Main File - xSE PluginPreloader F4 0.2.5.1 using the Manual Download button
  2. Download and extract the archive, then paste the 2 files in your game's Root folder
Allows F4SE plugins to be loaded before the game initializes.

Address Library for F4SE Plugins

Installation instructions:

  • Main File - Address Library
Resource required for other F4SE plugins.
After installing a mod, make sure you check the box next to it in the left pane of MO2 to enable it.

Buffout 4

Installation instructions:

  • Main File - Buffout 4
Fixes numerous bugs within the game's engine.

Shadow Boost

Installation instructions:

  • Main File - ShadowBoostFO4
  • Additionally install the Shadow Boost INI (Install manually in MO2)
Adjusts quality of godrays and distance of shadows, LOD and grass dynamically.

High FPS Physics Fix

Installation instructions:

  • Main File - High FPS Physics Fix
  • Once the mod has been installed, double-click on it in the left pane of MO2
  • In the INI-Files tab, select F4SE/Plugins/HighFPSPhysicsFix.ini
  • Change the following options:
    • Set DisableBlackLoadingScreens to true (line 37)
    • Set DisableAnimationOnLoadingScreens to true (line 44)
    • Set InGameFPS to a value you want the game FPS to be locked to (line 58)
      • If you set InGameFPS to a value higher than 60:
      • Set PipBoyFPS to 60.0 (line 79)
      • Set BudgetMaxFPS to the same value as InGameFPS (line 283)
      • Set DynamicUpdateBudget to true (line 264)
All-in-one solution for smooth FPS-unlocked gameplay, mitigates stutter, and reduces load times.

If you don't mind something more complex you can find a superior method to lock your FPS on this guide by Wall_SoGB.
