Utilities

Make sure the The Midnight Ride profile is active before proceeding.

Creating a Separator in MO2
  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Utilities.

Fallout 4 Script Extender

Installation instructions:

  1. Click the topmost download link
  2. Extract the contents of the folder from the downloaded archive to the game's Root folder
Make sure you are extracting all of the files inside of the f4se_x_y_z folder, not just the folder itself.
If you do not know what the Root folder is, read the Key Terminology section from the Initial Setup page.

Extends the scripting capabilities of the game.

In order for the script extender to work, you need to launch the game using the f4se_loader.exe file. It will automatically be added to Mod Organizer 2 upon restarting it.

PluginPreloader

Installation instructions:

  1. Download the Main File - xSE PluginPreloader F4 0.2.5.1 using the Manual Download button
  2. Download and extract the archive, then paste the 2 files in your game's Root folder

Allows F4SE plugins to be loaded before the game initializes.

High FPS Physics Fix

Installation instructions:

  1. Main File - High FPS Physics Fix
  2. Once the mod has been installed, double-click on it in the left pane of MO2
  3. In the INI-Files tab, select F4SE/Plugins/HighFPSPhysicsFix.ini
  4. Change the following options:
    • Set DisableBlackLoadingScreens= to true (line 37)
    • Set DisableAnimationOnLoadingScreens= to true (line 44)
    • Set InGameFPS to a value you want the game FPS to be locked to (line 58)
If you don't mind something more complex you can find a superior method to lock your FPS on this guide by Wall_SoGB.

All-in-one solution for smooth FPS-unlocked gameplay, mitigates stutter, and reduces load times.
