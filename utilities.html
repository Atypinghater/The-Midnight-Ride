Utilities

Make sure the The Midnight Ride profile is active before proceeding.

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Utilities.

F4SE

Installation instructions:

  1. Main Files - Fallout 4 Script Extender (F4SE) (Manual Download)
  2. From the downloaded archive, extract everything to the game's Root folder
If you do not know what the Root folder is, read the Key Terminology section from the Setup page.

Installation instructions:

xSE PluginPreloader F4

Installation instructions:

  • Main Files - xSE PluginPreloader F4 0.2.5.1 (Manual Download)

High FPS Physics Fix

Installation instructions:

  • Main Files - High FPS Physics Fix
  • Once the mod has been installed, double-click on it in the left pane of MO2
  • In the INI-Files tab, select F4SE/Plugins/HighFPSPhysicsFix.ini
  • Change the following options:
    • Set DisableBlackLoadingScreens to true (line 37)
    • Set DisableAnimationOnLoadingScreens to true (line 44)
    • Set InGameFPS to a value you want the game FPS to be locked to (line 58)
      • If you set InGameFPS to a value higher than 60:
      • Set BudgetMaxFPS to the same value as InGameFPS (line 283)
      • Set DynamicUpdateBudget to true (line 264)
All-in-one solution for smooth FPS-unlocked gameplay, mitigates stutter, and reduces load times.
