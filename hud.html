User Interface

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator UI.

Extended Dialogue Interface

Installation instructions:

  • Main File - Extended Dialogue Interface 1.3.6
Removes the hardcoded 4-option limit on dialogue and adds full engine support for any number of player dialogue options.

Mod Configuration Menu

Installation instructions:

  • Main File - Mod Configuration Menu 1.39
Adds an easy way to configure mods through the in-game pause menu.

HUDFramework

Installation instructions:

  • Main File - HUDFramework 1.0f
HUDFramework is a UI framework that makes it possible for mods to add new UI elements to the HUD.

Power Armor HoloHUD

Installation instructions:

  • Main File - PA HoloHUD
Cleans up the HUD while in power armor.
If you use an ultrawide aspect ratio, download and install the PA HoloHUD - Ultrawide file instead.

FallUI - Inventory

Installation instructions:

  • Main File - FallUI - Inventory
  • You will be prompted with a FOMOD installer, uncheck the Disable PipboyFX option, then click Next twice and Install
Optimized inventory interface for PC.

FallUI - Confirm Boxes

Installation instructions:

  • Main File - FallUI - Confirm Boxes
  • You will be prompted with a FOMOD installer, click Next and select the style you prefer, then Install
Overhaul of all confirm box interfaces.

FallUI - Sleep and Wait

Installation instructions:

  • Main File - FallUI - Sleep and Wait
Overhaul of the sleeping and waiting menu.

FallUI - Workbench

Installation instructions:

  • Main File - FallUI - Workbench
  • You will be prompted with a FOMOD installer, click Install
Optimized workbench interface for PC.

FallUI - Map

Installation instructions:

  • Main File - FallUI - Map
  • You will be prompted with a FOMOD installer, click Next and then Install
Overhaul of the map menu.

FallUI - HUD

Installation instructions:

  • Main File - FallUI - HUD
  • You will be prompted with a FOMOD installer, click Install
A highly configurable HUD.

MCM Booster

Installation instructions:

  • Main File - MCM Booster
Reduces the MCM loading time.

MCM Settings Manager

Installation instructions:

  • Main File - MCM Settings Manager
Allows you to save, load, transfer, import and export all your MCM settings at once.

MCM Categorizer

Installation instructions:

  • Main File - MCM Categorizer
Allows you to add your own categories into MCM for easier navigation.
