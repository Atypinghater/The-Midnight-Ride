Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input HUD + UI into the pop-up and select OK

DEF_UI

Installation instructions:

  • Download both main files through MO2
  • Being the installation for the Def_ui 1.5.1 file normally, and you will be prompted with a FOMOD installer
  • Select the following options from it:
    • ×
    • If you are using an aspect ratio other than 16:9, you can select whatever aspect ratio you use in the FOMOD. However, this will require extra tweaking later
  • After the installation is complete, hide the following from the Def_ui 1.5.1 file:
    • interface/Translate_en.txt
      • If you do not know what "Hide the following" means, read the IMPORTANT: Mod Installation Advice in Mod Organizer 2 section from the Mod Organizer 2 page
  • Install the def_ui_translations 1.10.163.0 main file normally

- Modernizes and improves many aspects of the HUD and UI

DEF_UI Iconlibs Rescaled and Fixed

HUD Framework

Files to install normally:

  • Main file - HUDFramework 1.0f
  • Optional file - 01. DEF_UI Compatibility Patch

- Ensures HUD extensions work flawlessly with one another
NOTE: The DEF_UI patch on this page only works for 16:9 aspect ratios. If you use a different aspect ratio, you will need to use the Auto-Patcher optional file to make it compatible

Improved Map

Installation instructions:

  • Download the Improved Map with Visible Roads 2.0 main file, the IMVR - Far Harbor (Default) optional file, and the IMVR - Nuka World (Default) optional file
  • Being the installation for the Improved Map with Visible Roads 2.0 file normally, and you will be prompted with a FOMOD installer
  • Select the Default option in the FOMOD then select Install
  • Install the other two downloaded files normally

- Makes the road lines on the Pip-Boy map more visible

Small Map Markers

Installation instructions:

  • Download the main file through MO2 and begin the installation normally
  • You will be prompted with a FOMOD installer, in which you can select whatever size option you prefer (I use the 50% version) then select Install

- Shrinks the map marker icons to avoid clutter on the world map
