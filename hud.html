Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator HUD + UI

DEF_UI

Installation instructions:

  1. Download both the Def_ui 1.5.1 and def_ui_translations 1.10.163.0 main files
  2. Begin the installation of the Def_ui 1.5.1 file, and you will be prompted with a FOMOD Installer
  3. Select the following options:
If you are using an aspect ratio other than 16:9, select it in the FOMOD instead of 16:9
  1. After the installation is complete, double-click on the DEF_UI mod in the left pane of MO2
  2. In the Filetree tab, right-click on interface/Translate_en.txt and select Hide
  3. Install the def_ui_translations 1.10.163.0 file normally

- Modernizes and improves the user interface

DEF_UI Iconlibs Rescaled and Fixed

Installation instructions:

  • Main File - DEF_UI Iconlibs Rescaled and Fixed - Uncolored 1.6

- Improved icons for DEF_UI

HUDFramework

Installation instructions:

  • Main File - HUDFramework 1.0f
  • Optional File - 01. DEF_UI Compatibility Patch
The DEF_UI patch only works with 16:9 aspect ratio resolutions. If you use a different aspect ratio, you need to use the Optional File - 02. HUDFramework Auto-Patcher 1.0 to create your own patch.

- UI framework required for many other HUD mods

Yet Another HUD Preset

Installation instructions:

  • Main File - Yet Another HUD Preset 1.6
