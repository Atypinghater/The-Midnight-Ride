Visuals

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Visuals.

Commonwealth Climate

Installation instructions:

  • Main File - Commonwealth Climate
Makes Fallout 4 a little more bleak/post-apocalyptic without completely changing the game's aesthetics.

Laser Weapons 1st Person Reposition

Installation instructions:

  • Main File - Institute and standard lasers
Makes the laser weapons take up less space in 1st person.

No Holotape Tease

Installation instructions:

  • Main File - No Holotape Tease - Texture Replacement
Changes the color of some debris textures to prevent them from looking like holotapes.

Interiors Enhanced

Installation instructions:

  • Main File - Interiors Enhanced 2.0 - ALL IN ONE
  • Also get the UFO4P and LOST Audio Tweaks patches here (select the name that matches the file on Nexus when the installation prompt comes up)
Makes interiors darker and more atmospheric, without affecting performance.

No More Glowing Mouths

Installation instructions:

  • Main File - No More Glowing Mouths
Prevents mouths from glowing in dark lighting.

Remove Ironsight Blur

Installation instructions:

  • Main File - RemoveBlur
Removes the blurring effect in weapon sights.

Terrain Undersides

Installation instructions:

  • Main File - Terrain Undersides
Block sun rays shining through the ground.
