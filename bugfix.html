Bug Fixes

Creating a Separator in MO2
  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Bug Fixes.

Weapon Debris Crash Fix

Installation instructions:

  • Main File - Weapon Debris Crash Fix
NOTE: This mod is only needed for users with NVIDIA GPUs newer than Pascal (10 series).

Fixes crashes with the weapon debris setting enabled caused by different PhysX implementation with new NVIDIA GPUs.

Previsibines Repair Pack

Installation instructions:

  • Main File - PRP Full Stable (0.65.0)
  • You will be prompted with a FOMOD installer, in which you should just select PPF, PRP, PRP - Interiors Enhanced Patch and PRP - Clarity 4.3 Patch
PRP - Interiors Enhanced Patch and PRP - Clarity 4.3 Patch are patches for mods installed later in the Visuals section of the guide. If you don't plan on installing either one, don't select the patch.
IMPORTANT: It is very important that the PRP.esp file form this mod remains at the bottom-most mod spot in your load order, otherwise you will have Previs issues (any patch you select in this FOMOD will have to go below PRP.esp). The load order at the end of the guide will take care of this for you, but this is very important to know if you plan on adding any extra mods.

Greatly improves FPS throughout the entire game, including interior cells. Also includes many placement or mesh fixes not possible without regenerating previsibines.
