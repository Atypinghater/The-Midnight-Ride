Bug Fixes

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Bug Fixes

Unofficial Fallout 4 Patch

Installation instructions:

  • Main File - Unofficial Fallout 4 Patch 2.1.3

- Fixes thousands of bugs with both the base game and DLCs

Ownership Fixes

Installation instructions:

  • Main File - Ownership Fixes 1.3.1

Sprint Stutter Fix

Installation instructions:

Weapon Debris Crash Fix

NOTE: This mod is only needed for users with NVIDIA GPUs newer than Pascal.

Installation instructions:

  • Main File - Weapon Debris Crash Fix 1.2

- Fixes crashes with the weapon debris setting enabled caused by different PhysX implementation with new NVIDIA GPUs

Previsibines Repair Pack

Installation instructions:

  • Download the Main File PRP Full Stable 0.59.5 and begin the installation through MO2
  • You will be prompted with a FOMOD installer, in which you should just select the PPF, PRP, PRP - Interiors Enhanced Patch and PRP - Clarity 4.3 Patch options
NOTE: PRP - Interiors Enhanced Patch and PRP - Clarity 4.3 Patch are patches for mods installed later in the Visuals section of the guide. If you do not plan on instaling either one, don't select the patches.

- Greatly improves FPS throughout the entire game, including interior cells

IMPORTANT: It is very important that the PRP.esp file form this mod remains at the bottom-most mod spot in your load order (the *right* pane of MO2), otherwise you will have Previs issues. The load order at the end of the guide will take care of this for you, but this is very important to know if you plan on adding any extra mods.
