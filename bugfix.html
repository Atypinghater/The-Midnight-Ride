Bug Fixes

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Bug Fixes.

Unofficial Fallout 4 Patch

Installation instructions:

  • Main Files - Unofficial Fallout 4 Patch
Fixes thousands of bugs with both the base game and DLCs.

Ownership Fixes

Installation instructions:

  • Main Files - Ownership Fixes
Fixes incorrect or missing ownership data assigned to junk and containers.

Community Fixes Merged

Installation instructions:

  • Main Files - Community Fixes Merged
A collection of fixes made by the community over the years with open permissions.

The Midnight Ride - Glitchfinder All-In-One

Installation instructions:

  • Main Files - The Midnight Ride - Glitchfinder All-In-One
Merged version of bug fixes created by Glitchfinder, specifically made for The Midnight Ride.

Buffout 4

Installation instructions:

  • Main Files - Buffout 4
Fixes engine bugs and adds a crash logger.

Outfit ReDress Fix

Installation instructions:

  • Main Files - Outfit ReDress Fix
Engine bug fix to prevent NPCs from defaulting their outfits.

Interior NavCut Fix

Installation instructions:

  • Main Files - Interior NavCut Fix
Fixes the engine bug that causes workshop navmesh cuts to persist throughout all interior cells.

Sprint Stutter Fix

Installation instructions:

  • Main Files - Sprint Stuttering Fix (first file)
Stops the camera from stuttering while sprinting on uneven surfaces.

Weapon Debris Crash Fix

Installation instructions:

  • Main Files - Weapon Debris Crash Fix
This mod is only needed for users with NVIDIA GPUs newer than Pascal (10 series).
Fixes crashes with the weapon debris setting enabled caused by different PhysX implementation with new NVIDIA GPUs.

Previsibines Repair Pack

Installation instructions:

  • Main Files - PRP Main
  • Main Files - PRP Updates
  • You will be prompted with a FOMOD installer, here select English, then PPF and PRP
Greatly improves FPS throughout the entire game, including interior cells. Also includes many placement or mesh fixes not possible without regenerating previsibines.

Wetness Shader Fix

Installation instructions:

  • Main Files - Wetness Shader Fix
Fixes the exaggerated shine of specific objects while it's raining.

Fixed Gobo Effects

Installation instructions:

  • Main Files - Fixed Gobo Effects
Fixes artefacts, pixelation and color banding in textures that are used for special light effects.

Long Save Bug Fix

Installation instructions:

  • Main Files - GCBugFix Release 0.1
A bug fix for one of the common causes of long save/autosave times.

MGEF Condition Bug Fix

Installation instructions:

  • Main Files - MGEF Condition Bug Fix
Fixes the magic effect condition bug that happens when you have a perk/ability for too long.

3rd Person Behavior Fixes

Installation instructions:

  • Main Files - 3rd Person Behavior Fixes
Fixes some glitches related to 3rd person behavior.

Jet Script Bug Fix

Installation instructions:

  • Main Files - Jet Script Bugfix
Fixes an issue with Jet that caused your thirst to go down if the chem was used by any NPC.

Legendary Drop Bug Fix

Installation instructions:

  • Main Files - Legendary Drop Bugfix
Fixes a bug, which causes legendary enemies not to generate and drop a legendary item.

Armor Penetration Bug Fix - F4SE

Installation instructions:

  • Main Files - Armor Penetration Bug Fix - F4SE
Prevents incorrect stat overwriting when unequipping or changing weapons.

Reload Fix

Installation instructions:

  • Main Files - Reload Fix
Fixes some reloading issues.

Power Armor Ownership Fix

Installation instructions:

  • Optional Files - PA Ownership Fix (with No PA Battery Pathing)
When you take over a Power Armor that belonged to someone else and insert a fusion core, it will no longer be considered stolen. Also stops you from having to path all the way around power armor when putting in a battery, as this can often result in getting stuck in terrain.

Far Harbor Marine Armor Boots Fix

Installation instructions:

  • Main Files - Far Harbor Boots
Restores the original Far Harbor Marine boots.

Marine Combat Armor Material Fix

Installation instructions:

  • Main Files - PBR
Adds missing material properties to the Marine Combat Armor.

Marine Wet suit Material Fix

Installation instructions:

  • Main Files - PBR
Adds missing material properties to the Marine Wet Suit.
