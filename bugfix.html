Bug Fixes and Quality of Life Improvements

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Bug Fix + QOL

Unofficial Fallout 4 Patch

Installation instructions:

  • Main File - Unofficial Fallout 4 Patch 2.1.2b

- Fixes thousands of bugs with both the base game and DLCs

Mod Configuration Menu

Installation instructions:

  • Main File - Mod Configuration Menu 1.39

- Adds an easy way to configure mods through the in-game pause menu

Boston FPS Fix

Installation instructions:

  • Main File - Boston FPS Fix - AIO .32
  • Optional File - PaladinBaileyRedRocket .1

- Greatly improves FPS in downtown Boston and the Concord area by safely rebuilding pre-combine data

Unofficial Fallout 4 Patch - Boston FPS Fix Patch

Installation instructions:

  • Download the mod using the download button pictured below
  • Finish the install normally through MO2

If you do not know how to download mods that do not have a Mod Manager Download button, read the Mod Installation Advice section from the MO2 page

- Forwards fixes from UF4P that were overwritten by Boston FPS Fix

Bullet Counted Reload System

Installation instructions:

  • Main File - Bullet Counted Reload - v.1.05b
  • Optional File - BCR- Lever Action Rifle v1.02

- Fixes the infamous reload bug with lever action weapons

Flicker Fixer

Installation instructions:

  • Main File - Flicker Fixer ESL v5.6

- Fixes many LOD flickering issues (also known as z-fighting) without modifying any pre-combine data

Canary Save File Monitor

Installation instructions:

  • Main File - Canary v 1.0.0

- Monitors your save file and warns you if it detects any irreversible data loss

Raw Input

Installation instructions:

  1. Download the Main File - RAW INPUT - NMM Installer 2.4 and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, where you should select the ADS sensitivity percent you prefer
    • This is completely up to personal preference, but I use the 100% option

- Disables mouse acceleration, normalizes sensitivity, and more input-related fixes

No Aggro Impact Landing

Installation instructions:

  • Main File - No Aggro Impact Landing 1.0

- Prevents the impact landing in Power Armor from turning friendly NPCs hostile

Unlimited Survival Mode

Installation instructions:

  • Main File - Unlimited Survival Mode - F4SE v1.4.0

- Restores the game features disabled in Survival mode, such as fast travel and saving. It's all fully configurable and can be skipped without problems if you consider it "cheaty"
