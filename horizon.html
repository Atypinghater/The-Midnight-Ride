Horizon

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Horizon.

Horizon

Installation instructions:

  • Main File - Horizon 1.8.7
  • You will be prompted with a FOMOD installer, click DLC (All-in-one), click Next, if you're interested in the hardest difficulty, click Desolation Mode (Hardcore), then click Next twice and then Install
  • Optional File - Horizon v1.8 Fixes ESP
Comprehensive Game Overhaul.

Blur Removal - Fix

Installation instructions:

  • Get the patch from HDPR and install the mod through MO2
Fixes a corrupted IMAD in the Blur Removal addon.
Only install this if you chose one of the Blur Removal plugins in the Horizon FOMOD.
HDPR: In the Discord Patches tab, use Ctrl+F to find the patch.

Load Order

Installation instructions:

  1. Download the file manually
  2. Once it has finished downloading, click the button at the top of MO2 and select Open Profile folder
  3. Extract the loadorder.txt from the downloaded archive to the folder you opened and overwrite when prompted
  4. Once the file is extracted, click F5 while you are on MO2 without any menu open
Automatically ensures your load order is correct for the guide. This will still work if you didn't install certain mods, but if you added your own mods they will be sorted to the bottom of the load order and need to be manually moved back to their proper position. If you do not know how to check where any mods you have added should go in the load order, it may be best to not add any mods not in the guide.
