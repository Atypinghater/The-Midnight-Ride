Wabbajack

The Wabbajack The Midnight Ride modlist will allow you to quickly and easily install the same modlist that the full manual guide (TMR Extended included) would result in.

Introduction and Setup

Read the Introduction first, then proceed to the Setup page. Once you're done, keep following this page.

Installation

  1. Download the latest Wabbajack.
  2. Create a new folder anywhere outside of any default Windows folders.
  3. Place the downloaded Wabbajack.exe in this folder, then run it.
  4. Enable Show Unofficial Lists, then locate The Midnight Ride in the gallery and click the download button in the corner.
  5. In Installation Location select an empty folder that is NOT the following:
    • The Steam folder.
    • Any default Windows folders.
    • The game folder.
    • The folder where you put Wabbajack.exe.
  6. Begin the installation.
  7. Accept the Nexus Mods API request.
  8. If you are not a Premium user you will need to manually click download for each mod.
  9. Once complete, the installation will show a green Installation Complete screen.
    • If you see a red Installation Failed screen, please join the ModdingLinked Discord server for support.

Post-Install Steps

  1. In your Installation Location, open Mod Organizer 2.
    2. You can skip steps 2 and 3 if you are ok with the game running at 60 FPS.
  2. Expand the Utilities separator, then double click High FPS Physics Fix and in the Filetree section, open HighFPSPhysicsFix.ini.
  3. Configure it according to the instructions here.
  4. Now you can play the game through the F4SE option in the top right:

Updating the List

Reinstall the list from the Wabbajack tool and enable the "Overwrite Installation" checkbox.
