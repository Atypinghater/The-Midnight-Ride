Crafting Highlight Fix

Installation instructions:

  • Download the main file manually
  • Extract the Data folder from the archive to the game's Root folder

- Replaces the full opacity item highlight during crafting with a much more clear outline

Weapon Debris Crash Fix

Installation instructions:

  • Download the main file manually
  • Extract the F4SE folder from the archive to the game's Data folder

- Fixes a crashing issue on NVIDIA RTX-enabled GPUs with the Weapon Debris setting enabled in the launcher
NOTE: This is only needed if you have an RTX card and have the aforementioned setting enabled (it is disabled by default on the High and Ultra launcher presets)

Place Everywhere

Installation instructions:

  • Download both the main file and the Place.ini optional file manually
  • From the main file archive, extract the F4SE folder to the game's Data folder
  • Repeat for the optional file

- Allows you to build settlement objects anywhere within the settlement's boundaries

Better Console

Installation instructions:

  • Download the main file manually
  • Extract the Data folder from the archive to the game's Root folder

- Adds extra info and functionality to the in-game console
