Tweaking the Fallout4Custom.ini:

  • Make sure the The Midnight Ride profile is active
  • Click the button on the top bar of MO2 and select INI Editor
  • Navigate to the Fallout4Custom.ini tab, which should be blank
  • Paste the following into the box:
      • [General]
      sStartingConsoleCommand=bat TMR

      [Display]
      fBlendSplitDirShadow=96
      iDirShadowSplits=3
      iMaxFocusShadows=3
      iMaxFocusShadowsDialogue=3
      uiOrthoShadowFilter=2

      [Imagespace]
      bMBEnable=0
      bDoRadialBlur=0

      [NVFlex]
      bNVFlexEnable=0

      [Archive]
      bInvalidateOlderFiles=1
      sResourceDataDirsFinal=
  • Repeat for the Testing profile
Tweak Effect
sStartingConsoleCommand=bat TMR Makes the game run a .BAT file that tweaks various settings upon starting the game (we will create the .BAT next)
fBlendSplitDirShadow=96
iDirShadowSplits=3
iMaxFocusShadows=3
iMaxFocusShadowsDialogue=3
uiOrthoShadowFilter=2
Tweaks various settings related to shadows to greatly increase performance without much visual decrease
bMBEnable=0 Disables motion blur
bDoRadialBlur=0 Disables radial blur (blur effects upon getting hit)
bNVFlexEnable=0 Disables NVIDIA's weapon debris option as its only available on RTX cards, and even on RTX cards it will cause crashes and potentially have a medium performance hit in heavy combat situations
bInvalidateOlderFiles=1
sResourceDataDirsFinal=
Allows loose files from mods to be loaded

Creating a .BAT File:

  • Download the .BAT file from here
  • Extract the .txt file from the archive to the game's Root folder
    • If you do not know what the Root folder is, read the Common Terminology section from the Installation page
Tweak Effect
gr quality 3
gr grid 12
gr scale .8
gr maxcascade 1
Tweaks various settings related to god rays to greatly improve performance with no visual decrease. Settings taken from Ultra Quality God Rays Performance Fix
cl rim 0.0025 Lowers the amount of light that radiates of all characters. Settings taken from Disable Character Light
fov 90 90 Sets the field of view to a much more comfortable setting
setgs iTerminalDisplayRate 1500 Speeds up the rate at which terminals display text. Settings taken from Faster Terminal Displays
setgs fWorkshopWireMaxLength 2200 Increases the length that settlement power lines can extend. Settings taken from Longer Power Lines
